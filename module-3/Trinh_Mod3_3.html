5 Things I've Learned Since the Course Started

  • Do Everything on Visual Studios, it makes everything so much easier
  • HTML may seem daunting at first, but once you understand, it isn't that bad. just a lot of opening and closing "<>"
  • With VSCode linking different files is so easy and a surefire way ensure that the proper file is being linked
  • While you can code and format text within the HTML file, it is a better practice to have the formatting in a ".css" file
  • Elements listed in the head are unseeable by the user, but give valuable information about the site.
